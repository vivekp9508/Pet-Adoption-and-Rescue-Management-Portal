# Pet Adoption and Rescue Management Portal

A Django-based web application to help reunite lost pets with their owners.

---

## Features

- User registration and authentication
- Pet listing and management
- Lost & found pet reporting
- REST API using Django REST Framework
- Admin dashboard for approval system

---

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite
- HTML / CSS

---

---

## Installation

1. Clone the repository
```
git clone https://github.com/vivekp9508/Pet-Adoption-and-Rescue-Management-Portal-.git
```
2. Install dependencies
```
pip install -r requirements.txt
```
3. Run migrations
```
python manage.py migrate
```
4. Run server
```
python manage.py runserver
```
